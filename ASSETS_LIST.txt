PROMPT OPENAI
==========================================

Create a 2D digital illustration of a fantasy monster inspired by classic Dungeon Master style.

IMPORTANT CONSTRAINTS (STRICT):
- Full body creature, front view
- Centered horizontally
- Bottom of the creature MUST touch the bottom of the canvas (no gap)
- Same scale and proportions as the reference image
- Creature must fit within the same visual size and framing as the reference
- No perspective distortion
- No change in style between images (must match previous monsters)

STYLE:
- Detailed fantasy painted style (NOT pixel art, NOT retro, NOT sprite style)
- Clean outlines, readable shapes
- Medieval dark fantasy aesthetic
- Slightly stylized but grounded

BACKGROUND:
- Pure white background (#FFFFFF)
- No shadow
- No environment
- No extra elements

ANIMATION VARIANT:
- If "idle": neutral pose
- If "walk": very subtle movement (small pose variation only, no displacement)
- If "attack": more dynamic pose, creature slightly projected forward (but same scale and base position)

CONSISTENCY:
- Must match the exact visual style, proportions, and framing of the reference image
- Designed to be used in a sprite animation sequence

OUTPUT:
- Single creature only
- No cropping
- No margins below the feet/body

Use this image as strict reference for proportions, scale, framing, and style.
